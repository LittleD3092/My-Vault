$$e^{ix} = \cos x+i\sin x$$

$$\sin x = \frac{ e^{ ix } - e^{ -ix } }{ 2i }$$

$$\cos x = \frac{ e^{ ix } + e^{ -ix } }{ 2 }$$

when $x = \pi$

$$e^{ i\pi } + 1 = 0$$
