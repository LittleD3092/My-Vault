# 微處理機 LAB 2.2

## PART 1.

### 1. Moder 設定

```c
RCC -> AHB2ENR |= RCC_AHB2ENR_GPIOBEN;

// 0b11111111_11111111_11111111_00000000
gpio -> MODER &= 0xFFFFFF00;    

// 0b00000000_00000000_00000000_01010101
gpio -> MODER |= 0x55;
```
