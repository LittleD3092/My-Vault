$$\Gamma(x) = \int_{0}^{\infty}t^{x - 1}e^{-t}dt$$

# 性質

## 1.階乘

$$\Gamma(n + 1) = n! \qquad n \; \text{is a positive integer}$$
$$\Gamma(1) = 0! = 1$$

## 2.提出

$$\Gamma (x + 1) = x\Gamma (x)$$

## 3.負數或零

$$\Gamma (n) \rightarrow \infty \qquad n \;\text{is a negative integer or}\; n = 0$$

## 4.分數

$$\Gamma(\frac{1}{2}) = \sqrt{\pi}$$
$$\Gamma(-\frac{1}{2}) = -2\sqrt{\pi}$$
