Polytropic: $P\nu^n = \text{ constant }$
$$\implies P = c\nu^{-n}$$

$$W = \int_1^2Pd\nu = \int_1^2 c\nu^{ -n }d\nu = c \cdot \frac{ \nu_2^{ -n + 1 } - \nu_1^{ -n + 1 } }{ -n + 1 } = \frac{ P_2\nu_2 - P_1\nu_1 }{ 1 - n }$$

$$\rightarrow \text{ For ideal gas: } \frac{ mR( T_2 - T_1 ) }{1 - n}$$
