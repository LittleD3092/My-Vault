把 First Law 的公式變成 rate function

$$\text{ First Law } \implies E = U + KE + PE = Q - W$$

$$\implies \frac{ dE }{ dt } = \frac{ dU }{ dt } + \frac{ d(KE) }{ dt } + \frac{ d(PE) }{ dt } = \frac{ dQ }{ dt } - \frac{ dW }{ dt }$$

$$\implies \dot{ E } = \dot{ Q } - \dot{ W }$$
$\dot{ Q }$: heat transfer rate
$\dot{ W }$: power
