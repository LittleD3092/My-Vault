# 椅子

椅子是用於屁股跟地板之間的關係

現代主義：簡單易生產並且具有功能性，可能容易收納，有可能會處理跟其他椅子的關係。有些也可能用廉價材料製作，或者用一體成形(如彎管、3d列印等)。

# 社會住宅

# 門把

# 空間的態度

# 空間的分隔

有些建築師會把空間的分隔界定的不明顯，例如室內室外沒有牆壁，階梯不同階有不同功能

35cm 是一個有任何可能性的距離，一個有 35cm 的立方體可以是座位，可以是床，也可以是階梯
