# Central Impact & Oblique Impact

| Central Impact             | Oblique Impact                         |
| -------------------------- | -------------------------------------- |
| $$\vec{v}_A // \vec{v}_B$$ | $$\vec{v}_A \bcancel{ // } \vec{v}_B$$ | 

# Coefficient of Restitution

$$e = \frac{ (v_B)_2 - (v_A)_2 }{ (v_A)_1 - (v_B)_1 }$$
