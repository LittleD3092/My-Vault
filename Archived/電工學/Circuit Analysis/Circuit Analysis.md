# Notation

## $v-t,\ i-t$ [[Curve]]

- 一個零件的電壓與電流用小寫 $v,\ i$ 代表，零件符號用大寫 $A,\ B$ 代表，如 $v_A,\ i_A$
- 方均根通常省略不寫，看到的 $I$ 或 $V$ 都是方均根的電壓、電流

## $i-v$ [[Curve]]

- 以 $v$ 為橫軸， $i$ 為縱軸
