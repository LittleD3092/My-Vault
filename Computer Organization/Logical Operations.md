Logical operations include shifting bits and bitwise operations. The following table shows some of them.

| Logical Operations | C Operators | MIPS Instructions |
| ------------------ | ----------- | ----------------- |
| Shift left         | `<<`        | `sll`             |
| Shift right        | `>>`        | `srl`             |
| Bit-by-bit AND     | `&`         | `and`, `andi`     |
| Bit-by-bit OR      | `|`         | `or`, `ori`       |
| Bit-by-bit NOT     | `~`         | `nor`             |
